Welcome to SouthernRust

Built for the Southern Community.

SouthernRust is a Rust Console Edition server built specifically for players in New Zealand and Australia. We focus on fair competition, active administration and a strong community.

No massive clans. No unchecked offline destruction. Just strong bases, competitive fights and proper console Rust.

Keep the server competitive, respectful and fair for everyone.

No Cheating
